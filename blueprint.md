# Luna Place - Personal Portfolio & Snake Game

## Overview

A personal portfolio website to showcase the user's skills, with an integrated, playable snake game as a fun, interactive element.

## Style, Design, and Features

### Design
*   **Style:** Modern dark theme inspired by retro arcade games, with vibrant accent colors.
*   **Layout:** Responsive design for both desktop and mobile.
*   **Effects:** Subtle animations and "glow" effects on interactive elements.
*   **Typography:** Clean, readable fonts with clear visual hierarchy.

### Features
*   **Header:** Displays the user's name/logo and navigation links ("About", "Game").
*   **About Section:** A section for a user biography and photo.
*   **Snake Game:**
    *   Fully functional snake game controlled by arrow keys.
    *   Score tracking.
*   **Footer:** Contains social media links or contact information.
